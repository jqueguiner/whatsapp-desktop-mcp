WhatsApp MCP

This installer places the whatsapp-mcp launcher at /usr/local/bin/whatsapp-mcp and a self-contained Python runtime at /usr/local/lib/whatsapp-mcp/.

After installation, add whatsapp-mcp to your Claude Desktop configuration and grant the three required macOS permissions (Full Disk Access, Accessibility, Automation).

See the project README at https://github.com/gladia/whatsapp-mcp for setup details.
